# Devops

Prerequisite
  * Vagrant 1.6.3+
  * Ansible 1.7+

To install the development environment run.

  * git clone https://github.com/opendroplet/droplet-app.git
  * cd droplet-app
  * vagrant up
  * vagrant provision
